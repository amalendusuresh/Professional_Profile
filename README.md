**Stack:** Solidity · Hardhat · ERC-20

Complete DEX implementation with Factory + Pair + Router architecture using the constant-product AMM (`x * y = k`).
**Stack:** Solidity · Uniswap V2 pattern · AMM

### [Launchpad — ICO + INO Platform](https://github.com/amalendusuresh/Launchpad)
Full token launchpad with ICO (token sales), INO (NFT sales), tier-based staking, and vesting. Modeled after DAO Maker / Polkastarter.
**Stack:** Solidity · ERC-20 · ERC-721 · Staking · Vesting

Production-ready TRC-721 NFT marketplace on TRON with escrow-based listings, TRX + TRC-20 (USDT) payments, and full backend/frontend stack.
**Stack:** Solidity · TRC-721 · TronBox · Node.js · React

### [ERC-20 Token Ecosystem](https://github.com/amalendusuresh/ERC20-Token)
DeFi token starter kit with Token, Staking, AMM, and Vesting contracts. Built with Hardhat Ignition and OpenZeppelin foundations.
**Stack:** Solidity · Hardhat · OpenZeppelin

### [Filecoin Staking DApp](https://github.com/amalendusuresh/Filecoin_Staking_DApp)

**Stack:** Solidity · Polygon · Hardhat · Node.js
